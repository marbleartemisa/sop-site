6.  Diferencia entre unidades 

• 

ft² (Square Feet) → Se usa para áreas (Countertop, Backsplash, Waterfall). 

• 

ft (Linear Feet) → Se usa para bordes lineales / miters. 

o 

Cada Finish Miter tiene un factor multiplicador, que representa el 
ancho del borde en pies, 

o 

Houzz Pro no multiplica automáticamente la longitud de la polyline por 
el factor para calcular la cantidad real de material (ft²) usado. 

Ejemplo en el Meeting: Finish Miter 3 

• 

Longitud dibujada: 3.77 ft 

• 

Factor del grupo (ancho): 0.25 ft 

• 

Cantidad de material = 3.77 × 0.25 = 0.94 ft² 

• 

✅ Houzz Pro muestra 0.94 ft (área real) 

 

 

 

 

 

  Traducción  simple:  Lo  que  dibujas  es  la  línea,  el  sistema  NO  aplica 

automáticamente el ancho para calcular cuánto material necesitas.

17. 

Resumen con Graficos tipo test 

 

 

1.  Miren el título: “Superficie plana” 

o 

Si  lo  que  estamos  midiendo  es  una 
superficie  visible  y  plana,  estamos 
hablando de área, no de borde. 

2.  Fíjense en los dibujos: counter, isla, 

backsplash, waterfall, apron. 

3.  Todos estos se miden como rectángulo 

(Rectangle en el software). 

4.  Si pueden tocar la superficie y verla 

completa, es área, no miters.

 Todo plano 

visible se mide en ft² 

 

1.  Miren el título: “Es un Miter?” 

o 

Solo los bordes con ángulo de 45° se 
miden como miters. 

2.  Pasos: 

o 

Dibuja solo la línea con la herramienta 
Polyline. 

o 

Activa el multiplicador (Multiplier) y 
escribe el ancho del borde. 

3.  Fórmula: Largo × Ancho = Área de 

material 

o 

Aunque diga FT, esto se considera área 
para corte de slab, no superficie plana. 

Nunca dibujen un miter como rectángulo. Eso 
es un error común.
