Guía Rápida de Precios — Iluminación de Piedras Translúcidas

Siga los pasos a continuación para obtener una cotización rápida. El precio final se obtiene sumando los precios individuales de cada área 
iluminada.

Paso 1 — Medir el área de la piedra o piedras
- Mida el ancho y largo de la piedra a iluminar en pulgadas.
- El ancho corresponde al lado más corto, y el largo al lado más largo de la superficie a iluminar.
- Si hay múltiples piedras o zonas, calcule el precio para cada una.

Paso 2 — Seleccionar el tipo de iluminación
- Unicolor (luz blanca o amarilla etc.)
- Multicolor (RGB o RGBW)

Paso 3 — Elegir el sistema de control
Para iluminación de Unicolor:
- Tabla 1 - On/Off: Encendido básico controlado por interruptor de pared que alimenta el sistema.
- Tabla 2 - Regulable (Dimmer): Permite ajustar la intensidad desde un dimmer en pared.
- Tabla 3 - Control Remoto: Encendido y ajuste la intensidad inalámbrico o desde el teléfono. (tambien aplica para Interruptor táctil oculto)

Para iluminación multicolor:
- Tabla 4 - MultiColor: Permite cambiar entre distintos colores. Incluye control remoto.

Paso 4 — Buscar el precio en la tabla correspondiente
- Ir a la tabla que corresponde al sistema de control elegido.
- Ubicar el ancho de la piedra en la columna izquierda.
- Ubicar el largo de la piedra en la fila superior.
- El cruce entre ambos da el precio total del sistema de iluminación para esa área.

Ejemplo práctico para  calcular el precio estimado de iluminación para una piedra de ónix
Dimensiones: 70" x 28" (usamos 28" como el ancho y 70" como el largo)
Tipo de iluminación: Unicolor, 6000K (luz blanca)
Sistema de control: Control remoto

- Ubique la tabla de precios correspondiente al "Sistema de un Solo Color - Control Remoto" (Tabla 3)
- Buscar la fila: el valor de ancho más cercano y por encima del ancho medido (Valor medido 28", valor cercano 28.25").
- Buscar la columna: el valor de largo más cercano y por encima del largo medido (Valor medido 70", valor cercano 79.5").
- Haciendo el cruce entre la fila y la columna seleccionadas, se ubica la celda con el precio correspondiente.
- El precio base del sistema es: $922.13

Notas:
- Los precios incluyen todos los componentes necesarios incluyendo unidades de LED, fuentes de poder, controladores, y cables.
- No incluye instalación.
- Para formas irregulares, use el ancho y largo máximos del área rectangular que contenga la figura.
- Para medidas mayores a las de la tabla, consulte directamente.
