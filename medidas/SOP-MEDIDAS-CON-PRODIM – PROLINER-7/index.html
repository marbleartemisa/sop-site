SOP-MEDIDAS CON PRODIM – PROLINER 7 

1.  Nombre del Procedimiento 

SOP-MED-003: SOP-MEDIDAS CON PRODIM – PROLINER 7  

2.  Objetivo 

Este procedimiento establece el método estándar para el levantamiento digital de 
medidas utilizando el sistema Prodim Proliner 7. 

El proceso aplica para: 

• 

Cocinas 

• 

Baños 

• 

Lavanderías 

• 

Countertops 

• 

Áreas de remodelación 

El sistema Proliner permite capturar formas complejas, rectas o curvas, generando 
automáticamente un archivo CAD DXF listo para diseño y fabricación. 

3.  Procedimiento 

3.1. 

 Preparación Antes de Medir 

Al llegar al sitio se debe realizar una verificación inicial. 

Confirmar: 

• 

Dirección del proyecto 

• 

Área exacta a medir 

• 

Que el finished floor esté instalado o confirmado 

• 

Ubicación de: 

o 

fregadero 

o 

electrodomésticos 

o 

gabinetes 

o 

paredes

2. Preparación del Equipo 

Posicionamiento del Proliner 

Colocar el equipo en una posición central 
del área a medir. 

Condiciones: 

• 

El cable del Proliner tiene un alcance 
máximo aproximado de 5 metros 

• 

Desde esa posición se debe alcanzar 
todo el perímetro 

Si el área es muy grande, se puede realizar más de una sesión de medición. 

Instalación del Equipo 

1.  Colocar el trípode en una superficie firme 
2.  Verificar que esté nivelado 
3.  Montar el Proliner en el trípode 
4.  Asegurarlo utilizando el tornillo de fijación 

Esto evita movimientos que afecten la precisión.

3. Encendido y Creación del Proyecto 

Encender el equipo. 

En el menú principal seleccionar:

 

 

Measure 

Luego elegir: 

Nuevo Proyecto 

Seleccionar: 

New 

Ingresar: 

• 

Nombre del proyecto 

• 

Cliente 

• 

Ubicación 

Luego presionar: 

Start 

 

Proyecto Existente 

Si el proyecto ya existe: 

1.  Buscar en la columna Project 
2.  Seleccionar el archivo en File 
3.  Abrir para: 

o 

agregar medición 

o 

editar medición 
existente

4. Definición del Plano de Medición 

Antes de medir el área se debe 
establecer el plano de referencia. 

Procedimiento:

 

 

1.  Medir mínimo 3 puntos base 
2.  Estos puntos crean el plano de 

proyección 

Este plano servirá como referencia 
para todas las mediciones del 
proyecto. 

 

5. Medición del Contorno Exterior 

Primero se debe medir el perímetro exterior del área. 

Esto incluye:

 

 

• 

Pared posterior 

• 

Pared izquierda 

• 

Pared derecha 

• 

Frente de gabinetes 

También se debe medir: 

• 

hasta dónde puede 
sobresalir el countertop 

• 

soportes estructurales 

 

 

  Importante: 

Medir la estructura del gabinete, no las puertas.

6. Medición del Contorno Interior 

Después del perímetro se deben medir todos los elementos que afectan el diseño del 
countertop. 

Esto incluye: 

Gabinetes 

Definir bordes y esquinas. 

Fregaderos 

Medir: 

• 

centro 

• 

bordes 

• 

dimensiones 

Faucets 

Ubicación del centro. 

Outlets 

Posición exacta respecto al borde o pared. 

Elementos adicionales 

Medir también: 

• 

registros 

• 

columnas 

• 

soportes 

• 

paredes irregulares 

• 

esquinas curvas 

Cada elemento debe definirse con puntos suficientes para describir su forma.

7. Edición del Archivo Digital 

Después de completar la medición: 

1.  Revisar los puntos capturados 
2.  Conectar los puntos para crear líneas 
3.  Definir contornos 
4.  Verificar dimensiones 

El archivo se genera en formato: 

DXF 

Este archivo se utiliza para: 

• 

diseño CAD 

• 

fabricación 

• 

corte CNC 

• 

control dimensional 

8. Capacidades del Sistema 

El Proliner permite medir: 

Formas 

• 

líneas rectas 

• 

curvas 

• 

ángulos complejos 

Posiciones 

• 

horizontal 

• 

vertical 

• 

inclinada 

Esto lo convierte en una herramienta ideal para templating de countertops.

9. Verificación Antes de Salir del Sitio 

Antes de abandonar el lugar se debe confirmar: 

✔ El perímetro completo fue medido 
✔ Todos los recortes están definidos 
✔ Los elementos interiores fueron incluidos 
✔ El archivo está guardado correctamente 

También se debe verificar que: 

• 

Las fotos fueron tomadas 

• 

Las notas del proyecto están completas 

10. Registro en el Sistema 

Al finalizar el trabajo: 

1.  Subir archivos al sistema 
2.  Cargar fotos 
3.  Registrar notas relevantes 

Luego marcar el Task como Completed en HouzPro. 

Checklist Rápido de Campo (Muy útil para técnicos) 

Antes de salir confirmar: 

☐ Fotos tomadas 
☐ Proyecto creado en Proliner 
☐ Plano base definido 
☐ Contorno exterior medido 
☐ Gabinetes medidos 
☐ Fregadero medido 
☐ Outlets registrados 
☐ Archivo DXF guardado 
☐ Información subida a HouzPro
