SOP TRANSFERENCIA DE INFORMACIÓN CON GOOGLE DRIVE 

 

1.  Nombre del Procedimiento 

SOP-VEN-006 TRANSFERENCIA DE INFORMACIÓN CON GOOGLE DRIVE  

2.  Objetivo 

Establecer el procedimiento estandarizado para la transferencia diaria de información 
entre Vendedores e Instaladores, utilizando Google Drive, garantizando que el 
instalador disponga de toda la información necesaria para ejecutar los trabajos 
asignados y que la empresa cuente con evidencia documental del trabajo realizado. 

3.  Alcance 

Este procedimiento aplica a: 

• 

Vendedores 

• 

Instaladores 

• 

Operador de IT 

• 

Administración / Recursos Humanos 

y regula el uso de Google Drive como plataforma de intercambio de información 
operativa diaria. 

4.  Descripción del sistema de directorios en Google Drive 

La empresa utiliza una estructura estandarizada de carpetas en Google Drive para 
organizar la información de las instalaciones. 

Cuenta principal utilizada:  

artemisacalendar@gmail.com

Directorio principal: 

INSTALACIONES 

Para cada instalador se crea un directorio individual: 

INSTALACIONES / INSTALADOR_X 

Dentro de cada instalador existen dos carpetas principales: 

INSTALACIONES / INSTALADOR_X / DIA 
INSTALACIONES / INSTALADOR_X / BACKUP 

Dentro de DIA se crean las siguientes subcarpetas: 

PLANOS 
FOTOS 
REPORTES-PUNCHOUT

6.  Responsabilidades 

6.1. 

 Administración / Personal 

Cuando un nuevo instalador ingresa a la empresa:

 

 

1.  La Gerencia Administrativa solicita formalmente al Operador de IT la creación 

del directorio del nuevo instalador. 

2.  Debe proporcionar el correo electrónico del instalador. 
3.  Este correo será utilizado para compartir el acceso al directorio DIA. 

6.2. 

Operador de IT 

 

El operador de IT deberá: 

1.  Crear el árbol de directorios en el Google Drive corporativo. 
2.  Ubicarlo dentro de: 

INSTALACIONES 

3.  Crear el directorio del instalador: 

INSTALACIONES / INSTALADOR_X 

4.  Crear las subcarpetas:  DIA, BACKUP

8.  Procedimiento del Vendedor (Tarde anterior) 

Después de la reunión de programación de instalaciones, el vendedor responsable 
deberá preparar la información del instalador.

11. Documentación del trabajo 

Al finalizar el trabajo del día el instalador debe: 

1.  Abrir Google Drive. 
2.  Ir a: 

DIA / FOTOS 

3.  Utilizar el ícono de cámara para: 

• 

Tomar fotos 

• 

Grabar videos 

Los archivos se guardarán directamente en la carpeta correcta. 

11.1.  Reglas importantes 

No guardar fotos fuera de esta carpeta. 

Después de cada archivo: 

1.  Presionar NEXT 
2.  Presionar UPLOAD 

11.2.  Conexión a internet 

•  Si hay internet: 
•  El archivo se sube inmediatamente. 
•  Si no hay internet: 
•  El archivo quedará pendiente y se subirá automáticamente cuando haya 

conexión.

11.4.  Cierre del proceso 

Una vez finalizada la jornada: 

1.  El instalador habrá cargado fotos, videos y reportes. 
2.  El vendedor revisará la información. 
3.  La documentación será subida a Houzz Pro. 
4.  Los archivos se respaldarán en BACKUP. 

Con esto el ciclo queda listo para el siguiente día de trabajo.
